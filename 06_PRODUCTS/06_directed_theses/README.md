# 06_directed_theses

## Purpose

This folder contains the documentary basis for directed theses and supervised undergraduate research projects associated with the doctoral work:

**“Estudio de la velocidad de viento e inclusión de parámetros físicos para la predicción de energía eléctrica producida por fuentes eólicas”.**

The purpose of this folder is to document the academic projection of the doctoral thesis into student research, applied technological development, renewable-energy applications, scientific information systems, and decision-support platforms.

This folder is part of:

```text
06_PRODUCTS/
```

and should be interpreted as the supervised academic production layer of the doctoral repository.
