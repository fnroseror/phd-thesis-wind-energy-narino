# 04_dashboard

## Purpose

This folder contains the documentary basis for the Power BI dashboard derived from the doctoral research:

**“Estudio de la velocidad de viento e inclusión de parámetros físicos para la predicción de energía eléctrica producida por fuentes eólicas”.**

The purpose of this folder is to document the applied visualization product associated with the doctoral thesis, including its public access link, current scope, scientific relationship with the research, and repository evidence.

This folder is part of:

```text
06_PRODUCTS/
```

and should be interpreted as the applied visualization and academic dissemination layer of the doctoral repository.

## 13. Data and publication caution

The dashboard is accessible through a public Power BI view link.

For this reason, the dashboard should only include data, summaries, indicators, or visualizations appropriate for public dissemination.

The dashboard should not expose:

- restricted raw data;
- personal data;
- private credentials;
- internal institutional data;
- confidential model files;
- private connection strings;
- non-authorized datasets;
- or sensitive information not intended for public release.

Repository interpretation:

```text
The dashboard is a public visualization product derived from the thesis.
The Power BI source file is not required in the public repository unless fully sanitized and authorized.
```

## 14. Source file caution

The `.pbix` source file is not required for this public repository.

If the `.pbix` file is ever considered for upload, it should first be reviewed to ensure that it does not contain:

- local file paths;

- hidden tables with sensitive content;
- internal comments;
- or non-public datasets.
