This folder contains the documentary basis for the Power BI dashboard derived from the doctoral thesis:

**“Estudio de la velocidad de viento e inclusión de parámetros físicos para la predicción de energía eléctrica producida por fuentes eólicas”.**

## 4. Relationship with the doctoral thesis

The dashboard is derived from the doctoral thesis and functions as an applied visualization layer for the scientific results.

The thesis develops a physical–statistical framework for:

- physical characterization of wind behavior;
- Wind Power Density construction;
- predictive modeling;
- uncertainty quantification;
- FNRR computation;
- free energy estimation;
- usable energy estimation;
- and regional energy projection.

The dashboard translates part of this framework into a visual and interactive product.

The relationship can be summarized as:

```text
doctoral thesis
→ physical–statistical results
→ organized repository evidence
→ Power BI dashboard
→ applied visualization and dissemination
```

## 5. Scientific function of the dashboard

The dashboard supports the visual interpretation of the doctoral research.

Its scientific and applied functions may include:

- communicating regional wind-energy potential;

- supporting interpretation of Wind Power Density;
- presenting predictive analytics outputs;

- communicating uncertainty-aware results;
- supporting FNRR-based interpretation;

## 10. Data and publication caution

The dashboard is publicly accessible through Power BI.

For this reason, the dashboard should use only data, summaries, indicators, or visualizations that are appropriate for public dissemination.

The repository should not expose:

- sensitive raw data;
- private credentials;
- internal connection strings;
- restricted institutional information;
- personal data;
- or non-authorized datasets.

The `.pbix` source file does not need to be uploaded to this public repository unless it is fully sanitized and authorized for public release.
