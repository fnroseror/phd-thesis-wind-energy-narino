\# Dashboard



This folder contains the documentary basis for the projected dashboard derived from the doctoral thesis and its applied dissemination potential.



\## Purpose

The purpose of this section is to document the dashboard as a projected applied product, even if its implementation is partial or still pending.



\## Main file

\- `dashboard\_status.md`



\## Scope

The dashboard status file should define:

\- intended function,

\- potential users,

\- core variables or outputs to be displayed,

\- relation to thesis results,

\- current implementation status.



\## Relationship with the thesis

This dashboard is conceived as an applied visualization and dissemination layer derived from the doctoral work on wind-energy estimation, uncertainty-aware forecasting, regional comparison, and usable energy interpretation.



\## Repository role

This folder functions as the documentary bridge between the scientific repository and a projected interactive or semi-interactive applied visualization product.
