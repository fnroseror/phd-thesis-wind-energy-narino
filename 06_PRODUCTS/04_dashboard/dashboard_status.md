# Dashboard Status

## Purpose

This file documents the current status of the Power BI dashboard derived from the doctoral thesis of **Favio Nicolás Rosero Rodríguez**.

The dashboard is associated with the thesis:

**“Estudio de la velocidad de viento e inclusión de parámetros físicos para la predicción de energía eléctrica producida por fuentes eólicas”.**

## 5. Relationship with the doctoral thesis

The dashboard is derived from the doctoral thesis and its associated repository.

The thesis provides the scientific framework:

```text
meteorological observations
→ physical characterization
→ Wind Power Density
→ predictive modeling
→ PI90 uncertainty
→ FNRR
→ free energy
→ usable energy
→ regional projection
```

The dashboard provides an applied visualization layer for communicating selected elements of this framework.

- personal data;
- private credentials;
- internal institutional data;
- confidential model files;
- or non-authorized information.

```text
The dashboard is a public visualization product derived from the thesis.
The Power BI source file is not required in the public repository unless fully sanitized.
```
