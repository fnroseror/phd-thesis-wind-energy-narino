# Dashboard Status

## Purpose

This file documents the current status of the Power BI dashboard derived from the doctoral research of **Favio Nicolás Rosero Rodríguez**.

The dashboard is associated with the doctoral thesis:

**“Estudio de la velocidad de viento e inclusión de parámetros físicos para la predicción de energía eléctrica producida por fuentes eólicas”.**

This document provides repository-level traceability for the dashboard as an applied visualization and dissemination product focused on the temporal behavior of the meteorological variables used in the doctoral research.

## 14. Publication caution

The dashboard is accessible through a public Power BI view link.

For that reason, the dashboard should only include data, summaries, indicators, or visualizations appropriate for public dissemination.

The dashboard should not expose:

- restricted raw data;
- personal data;
- private credentials;
- internal institutional data;
- confidential model files;
- private connection strings;
- non-authorized datasets;
- or sensitive information not intended for public release.

Repository interpretation:

```text
The dashboard is a public visualization product derived from the thesis.
The Power BI source file is not required in the public repository unless fully sanitized and authorized.
```
