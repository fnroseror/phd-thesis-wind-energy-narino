# 07_professorial_project

## Purpose

This folder contains the documentary basis for the professorial research project associated with the doctoral thesis:

**“Estudio de la velocidad de viento e inclusión de parámetros físicos para la predicción de energía eléctrica producida por fuentes eólicas”.**

The purpose of this folder is to document the institutional, academic, scientific, and research-line projection of the doctoral work through a formally approved professorial project within the Programa de Ingeniería de Sistemas, Universidad Mariana.

This folder is part of:

```text

```

and should be interpreted as the institutional research-project evidence layer of the doctoral repository.

## 9. Relationship with the doctoral thesis

This project is directly connected to the doctoral thesis because it shares the same central scientific problem and extends it into an institutional professorial research framework.

The doctoral thesis develops:

- physical characterization of wind behavior,
- Wind Power Density construction,
- predictive modeling,
- uncertainty quantification,
- FNRR,
- free energy,
- usable energy,
- and regional energy projection.

The professorial project institutionalizes and extends this research line within the Programa de Ingeniería de Sistemas.

The relationship can be summarized as:

```text
doctoral thesis
→ physical–statistical wind-energy framework
→ institutional professorial research project
→ academic and technological projection
```
