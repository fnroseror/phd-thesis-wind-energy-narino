# Execution Pipeline

This document describes the correct execution order of scripts used in the thesis.

## Step 1 – Data Cleaning
- Removal of duplicates
- Temporal consistency validation
- Controlled interpolation of missing values

## Step 2 – Feature Engineering
- Air density calculation
- Wind Power Density (WPD)
- Integrated Energy (Eh)

## Step 3 – Classical Modeling
- ARIMA / ARIMAX models

## Step 4 – Machine Learning
- Random Forest
- Gradient Boosting

## Step 5 – Deep Learning
- LSTM-based architectures
- Bayesian optimization

## Step 6 – Hybrid TDQ-PIESS Framework
- Physical-statistical integration
- Skill score optimization

## Step 7 – Export of Final Results
- Metrics
- Comparative tables
- Publication-ready figures
