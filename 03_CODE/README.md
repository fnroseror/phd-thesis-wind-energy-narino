# 04_CODIGO – Executable Scripts

This folder contains the finalized R scripts used in the PhD thesis.

## Script overview

- cap2_physical_spectral_analysis.R  
  Physical and spectral characterization of wind variables.

- cap3_classical_models.R  
  ARIMA / ARIMAX baseline models.

- cap3_ml_models.R  
  Machine learning models (RF / XGB).

- cap3_dl_lstm_bayes.R  
  Deep learning LSTM models with Bayesian optimization.

- cap3_tdq_piess_model.R  
  Hybrid TDQ–PIESS model and PI90 calibration.

- cap4_tdq_projection_2028.R  
  Final unified model and 2028 energy projection.
