# 05_RESULTADOS – Consolidated Experimental Results

This folder contains the master results table used in the thesis.

## File

MASTER_TABLA_RESULTADOS_CAP3_CAP4.csv

## Description

Each row corresponds to a trained model configuration for a given:

- Zone
- Forecast horizon
- Target variable (WPD or Eh)
- Model type (Classical / ML / DL / TDQ–PIESS)

## Metrics included

- RMSE
- MAE
- R²
- Skill vs Persistence
- PI90 coverage
- FNRR mean test value

Hyperparameters for DL models are included for full reproducibility.
