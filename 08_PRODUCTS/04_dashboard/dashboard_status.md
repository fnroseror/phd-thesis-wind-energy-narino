\# Dashboard Status



\## Projected product name

Wind-energy and uncertainty dashboard for regional analysis in Nariño



\## Intended function

The projected dashboard is intended to provide an organized visual layer for exploring the main outputs of the doctoral thesis, especially those related to regional wind-energy behavior, predictive performance, uncertainty, and usable energy interpretation.



\## Potential users

This projected dashboard may be useful for:

\- academic evaluators,

\- researchers,

\- institutional stakeholders,

\- students,

\- future collaborators interested in regional wind-energy analysis.



\## Core variables or outputs to be displayed

The dashboard is expected to display, at minimum:

\- wind power density summaries by zone,

\- model comparison results,

\- uncertainty-aware outputs,

\- PI90-related interpretation,

\- FNRR-related structural summaries,

\- distinction between free energy and usable energy,

\- selected figures and tables derived from the thesis.



\## Relationship with thesis results

The dashboard is not conceived as an independent product disconnected from the thesis.  

It is a visualization and interpretation layer derived from the scientific outputs already generated in the doctoral work.



\## Current implementation status

Projected product.  

The conceptual basis is identified, and the repository already contains the scientific outputs that could support future implementation.  

A complete interactive implementation is not declared as finished in the current repository version.



\## Value of documenting it now

Documenting the dashboard in the repository is useful because it:

\- preserves the applied vision of the doctoral work,

\- clarifies the potential translation of results into a usable interface,

\- supports future development without losing conceptual direction.



\## Final note

This file documents the current status and projected logic of the dashboard as a derivative product of the doctoral thesis.
