# 05_FIGURES

## Purpose

This folder contains the visual material associated with the doctoral thesis.

Figures are organized according to thesis chapter and supplementary support material.

---

## Structure

### `chapter_2/`
Figures associated with physical analysis and factor characterization.

### `chapter_3/`
Figures associated with predictive modeling and data-driven analysis.

### `chapter_4/`
Figures associated with the hybrid framework, FNRR logic, and integrated energy projection.

### `supplementary/`
Additional figures used as technical support outside the main thesis body.

---

## Scientific role

This folder separates visual evidence from executable code and tabular outputs, improving the readability and auditability of the repository.

---

## Closure criterion

This folder is considered complete when each figure is stored in the chapter to which it belongs and naming is consistent with thesis structure.
