# PhD Thesis – Wind Energy Forecasting in Nariño (2017–2022)

This repository contains the complete reproducible technical annexes of the doctoral research.

## Repository structure

01_ANEXOS → Written annexes (A–F)  
04_CODIGO → Final executable R scripts  
05_RESULTADOS → Consolidated master results table  
06_REPRODUCIBILIDAD → Environment, data contract, execution pipeline  

## Experimental scope

• Physical-spectral analysis (Chapter 2)  
• Classical baseline models (ARIMA/ARIMAX)  
• Machine Learning (RF / XGBoost)  
• Deep Learning (LSTM with Bayesian optimization)  
• Hybrid TDQ–PIESS framework  
• 2028 energy projection
